Book Store E‑Commerce App
A full‑stack MERN Book Store application where users can browse books, filter by genre, and place orders.
The project is DevOps‑ready and designed for easy integration with Docker, Kubernetes, and CI/CD pipelines.

Features:
User signup & login (JWT authentication)
Browse books and filter by genre
View book details
Add to cart & place orders (dummy checkout)
Admin: manage books, genres, and view orders

Tech Stack
Frontend: React
Backend: Node.js, Express
Database: MongoDB Atlas
Auth: JWT
DevOps: Docker, Kubernetes 

Run Backend
cd backend
npm install
npm start

Server runs at:
http://localhost:5000
